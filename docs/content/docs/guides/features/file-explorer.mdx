---
title: File Explorer
---

Peek behind any Library or Series and there's a file explorer scoped to just that entity, letting you traverse the files and directories that make it up without ever leaving Longbox. The entities that have a file explorer are:

- **Libraries**
- **Series**

## Accessing a File Explorer

<Callout title="Permission required">
	**HOUSE RULES** — This functionality is gated behind the `FILE_EXPLORER` user permission. To learn
	more about permissions, see the [Permissions](/docs/guides/access-control/permissions) guide.
</Callout>

Generally, you can access a file explorer by clicking on the `Files` tab in an entity's page. This will render the file explorer for that entity, which effectively is just sets the root directory of the file explorer to the entity's path.
