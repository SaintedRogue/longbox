Longbox can host your own book club — a standing invite to read something together. This guide
walks through how the feature works, what it can do, and how to set one up.

## » Setting up shop (basic requirements)

To set up a book club in Longbox, you'll need:

- A Longbox instance
- A managed user account (otherwise, it would be a club of one)

You don't even technically need any books on the server to start a book club, either, since you can create book club schedules with 'external' books.

## » Who's got the invite (how it works)

Book clubs in Longbox are effectively just a group of users that all have access to special pages and functionalities that are only available to members of a given book club. There can be multiple book clubs in a single Longbox instance, and users can be members of multiple book clubs.
