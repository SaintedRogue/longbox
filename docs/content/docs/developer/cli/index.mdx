The `longbox` CLI is the back-room counter — account chores and server tasks without the web app.

<Callout type="info">
	**FROM THE LONGBOX** — Every example in this guide assumes you are running the CLI tool from the
	same directory as the executable, and that it is named `longbox`. You will need to adjust the path
	to the executable accordingly if you are running it from a different location or if it has a
	different name
</Callout>

## » Working the counter (basic usage)

The CLI tool is exposed via a few subcommands using the same executable as the server, itself. To see the available subcommands, run:

```bash copy
./longbox --help
```

This will print the various subcommands and options available to you. To see more information about a specific subcommand, run:

```bash copy
./longbox <subcommand> --help
```
